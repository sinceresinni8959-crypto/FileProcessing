## Project Description
This project is a Windows Forms application for reading CSV files.

## Features
- Read large CSV files using StreamReader
- Partial loading (Start - End)
- Filter by file type
- Combined range loading and filtering

## Development Environment
- Visual Studio 2026
- C#
- .NET Windows Forms

## How to Run
1. Open the solution in Visual Studio.
2. Build and run the project.
3. Select a CSV file.
4. Enter Start, End, and Type (optional).
5. Click **Read as CSV**.
